Projeto desenvolvido como parte da disciplina de Banco de Dados. Esta aplicação utiliza tecnologias modernas no Front-end e Back-end para criar uma plataforma de gerenciamento de academia, onde é possível cadastrar, editar, deletar e visualizar treinos, alunos, professores, entre outros componentes de uma academia.

# *Tecnologias Utilizadas*

*Front-end*: Vue.js
*Back-end*: ASP.NET (C#)
